## 学習ステップ

### Phase 1: Basics (01_Basics)
- [ ] Installation (rustup, cargo)
- [ ] Variable Bindings and Mutability
- [ ] Data Types (Scalar, Compound)
- [ ] Functions and Control Flow
- [ ] **Ownership, References, and Borrowing** (重要)

### Phase 2: Intermediate (02_Intermediate)
- [ ] Structs and Enums (Pattern Matching)
- [ ] Modules and Crates
- [ ] Common Collections (Vector, String, Hash Map)
- [ ] Error Handling (Result, Option)

### Phase 3: Advanced (03_Advanced)
- [ ] Generics, Traits, and Lifetimes
- [ ] Writing Automated Tests
- [ ] Smart Pointers (Box, Rc, RefCell)
- [ ] Fearless Concurrency (Threads, Message Passing)

### Phase 4: Capstone Project (04_Project)
- [ ] AIなしで、基本的な HTTP サーバを `std::net` のみで実装。
- [ ] メモリ安全性を意識した、コマンドライン・ファイル検索ツール (grep クローン) の作成。
